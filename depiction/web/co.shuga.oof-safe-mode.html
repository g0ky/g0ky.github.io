Compatibility: This package is compatible with iOS iOS 8.0 to 13.0.

This replaces the stock safe mode icon (the "Sad iPhone") with the word "oof!"

Warning! This package is not compatible with any other mods for Safe Mode (such as SublimeSafeMode) and is recommended to be uninstalled while updating Tweak Injector. In addition, it must be removed prior to Tweak Injector updates to prevent ensure your backups are correct and up-to-date.

To switch from SublimeSafeMode: Please uninstall SublimeSafeMode, reinstall Tweak Injector (to properly restore SafeMode.app), and install this package.
