Compatibility: This package is compatible with iOS iOS 8.0 to 13.0.

elementary is a simplistic, monochrome, glyph-based theme. This version themes system icons.

Note: This is prerelease software and bugs are to be expected, as well as frequent updates. Do not install this if you want a stable theme until this is marked as stable.

Currently supports:

  • (Nearly) All Stock iOS Icons
  • Crucial Jailbreak-Related Icons (such as Filza, Cydia, and Electra)
  • Badges
  • Icon Masks
  • Chat Bubbles
